# aetherly-stream-proxy example

Minimal Next.js App Router app demoing the proxy. Paste an HLS or MP4 URL, optionally a `Referer` header, and play it through the proxy.

## Local

```bash
npm install
npm run dev
# → http://localhost:3000
```
